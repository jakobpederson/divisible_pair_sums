
You are given an array ar of integers and a positive integer k. Find and print the number of pairs where index(i) < index(j) and i + j is divisible by k.

Input Format

The first line contains 2 space-separated integers, n and k, respectively.
The second line contains n space-separated integers describing the respective values of ar.

Constraints

Output Format

Print the number of pairs where index(i) < index(j) and i + j is evenly divisible by k.

Sample Input
6 3
1 3 2 6 1 2

Sample Output
5
